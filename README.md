# Chapter-8
多媒体进阶-拍照录制

### 作业完成情况

基础功能完成，能实现点击PICTURE拍摄单张照片。点击RECORD开始拍摄视频，再次点击停止。点击FACING切换前后摄像头。点击ZOOM进行调焦。拍摄完成后自动开始播放，播放完成后自动恢复拍摄状态。
